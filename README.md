# OVEX

Convert currency with [OVEX](https://ovex.com/)

![Screenshot](/assets/images/screenshot.png)

## Installation

1. **Clone repository**

   ```bash
   git clone https://github.com/delvianv/ovex.git
   cd ovex
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run development server**

   ```bash
   npx expo start
   ```
